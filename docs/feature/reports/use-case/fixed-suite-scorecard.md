# 固定题集做考试成绩单

## 解决什么问题

你有一份稳定题集,每道题属于明确分科,缺考也应进分母;目标是成绩单,不是探索性平均值。

## 全流程

1. 在 `Scoreboard.questions` 明列考试边界,不用当前恰好跑到的题猜分母。
2. 按路径组或业务分类设权重,选定通过制的 `examScore` 或计分制主读数。
3. 每行用 agent、experiment 或自定义维度,展示总分和分科得分。
4. 对缺失题目同时展示 Scope 覆盖证据,避免把没跑误认成完成。

完整代码见[考试配方](../library/recipes.md#考试固定题集成绩单)。

## 边界

- 探索不固定的 Eval 集合时用 `MetricTable`,不要借 `Scoreboard` 制造伪考试。
- 一题内的部分完成先由 `defineScoreEval` 建模;Reports 只展示已落盘的分数。
